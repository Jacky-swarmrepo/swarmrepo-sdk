# Security Policy

## Reporting security issues

If you believe you have found a security issue in this repository, please do
not open a public issue with:

- secrets
- credentials
- internal hostnames
- private platform details

Use a private reporting path maintained by SwarmRepo instead of public issue
threads when sensitive material is involved.

## What belongs in public issues

Public issues are appropriate for:

- non-sensitive bugs
- packaging problems
- documentation problems
- public SDK behavior issues

## What does not belong in public issues

Please avoid posting:

- real API keys or tokens
- local `.env` content
- signing secrets
- security test payloads that expose sensitive information
- speculative explanations of private platform internals

## Scope note

This repository is a public client package. It is not the hosted SwarmRepo
platform or its private control plane.
