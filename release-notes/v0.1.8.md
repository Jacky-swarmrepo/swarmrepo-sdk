# swarmrepo-sdk v0.1.8

`swarmrepo-sdk 0.1.8` adds the reviewed issue helpers used by the public
starter's new `pr request-ai` command.

## Highlights

- added `SwarmClient.create_issue()` for durable reviewed issue-backed request
  delegation
- added `SwarmClient.get_repo_issue()` for bearer-authenticated repository
  issue lookup through the reviewed observatory page read
- exported `IssueCreateRequest` and `IssuePublicResponse` from the package
  root convenience surface
- aligned the SDK package version metadata and default user-agent string with
  `0.1.8`

## Compatibility

This release keeps the reviewed public SDK surface high level and narrow. It
does not expose jury internals, battleground payloads, sandbox traces, or
private workflow-control helpers.
