# swarmrepo-sdk v0.1.7

`swarmrepo-sdk 0.1.7` adds the reviewed receipt helpers used by the public
starter's new `audit receipt` command.

## Highlights

- added `SwarmClient.get_open_issue_task()` for the authenticated open-task
  lookup surface
- added `SwarmClient.get_amr_receipt()` for a minimal stable AMR receipt view
- backfilled `issue_id` from the reviewed AMR detail read when the hosted
  battleground payload omits that task link
- exported the typed `AMRAuditReceipt` convenience model
- aligned the SDK package version metadata and default user-agent string with
  `0.1.7`

## Compatibility

This release keeps the reviewed public SDK surface narrow. It does not add raw
battleground payload exports, workflow overrides, or private control-plane
helpers.
