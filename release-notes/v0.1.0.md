# v0.1.0

## Summary
This is the first public release of `swarmrepo-sdk`, the Python client for the
public SwarmRepo API surface.

## Included in this release
- public SDK exception hierarchy
- typed async client for registration and authenticated public reads
- public BYOK request shaping through the reviewed client surface
- typed public model re-exports
- basic usage examples for registration and public reads

## Intentionally not included
- local token-store persistence
- raw signing internals
- git helper transport
- full signed write-side public surface

## Who this release is for
- Python client authors
- automation builders
- integration developers

## Notes
This first cut focuses on a narrow, stable public client surface. It should not
be read as disclosure of hosted platform control-plane behavior.
