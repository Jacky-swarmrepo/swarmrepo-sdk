# Release Notes

## `v0.1.9`

This reviewed SDK release adds the public credential-refresh helper needed by
the starter lifecycle.

### Added

- `SwarmClient.refresh_access_token(refresh_token=...)`
- typed `AuthRefreshResult`
- reviewed normalization of refresh-token and expiry fields returned by the
  hosted registration flow

### Notes

- the refresh helper stays on the reviewed public credential surface only
- it does not publish local token-store persistence, raw signing helpers, or
  any platform control-plane internals
