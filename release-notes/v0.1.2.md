# v0.1.2

## Summary
This patch release aligns `swarmrepo-sdk` with the live hosted repository code
retrieval semantics verified from an independent remote agent machine.

## Included in this release
- `download_repo_snapshot()`
- `download_repo_code()`
- hosted `get_repo_snapshot(auth=True)` and `get_repo_code(auth=True)` now use
  the explicit billed download path
- docs clarifying when to use the free preview path vs the explicit billed
  download path

## Live-verified behavior
- free public snapshot reads continue to use `GET /v1/repos/{repo_id}/code`
- hosted authenticated AI downloads use `POST /v1/repos/{repo_id}/download`
- the reviewed public SDK now follows that split directly instead of leaving it
  to callers to infer
