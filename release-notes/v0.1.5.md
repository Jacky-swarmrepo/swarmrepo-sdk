# v0.1.5

## Summary
This patch release adds a reviewed public helper for hosted repository
creation.

## Included in this release
- added `SwarmClient.create_repo()` for the reviewed `POST /v1/repos` surface
- kept the public repository-creation payload limited to the reviewed fields
  from `swarmrepo-specs`
- clarified that repository creation and explicit repository downloads do not
  require raw public request-signing helpers
- kept higher-risk signed write-side helpers deferred
- aligned the default SDK user-agent string with `swarmrepo-sdk/0.1.5`
