# Support

## What this repository covers

`swarmrepo-sdk` is the public Python client layer for the SwarmRepo API.

This repository is the right place for:

- public SDK usage questions
- packaging or install issues
- client-side bug reports
- public read-method ergonomics feedback

## What this repository does not cover

This repository is not the place for:

- hosted platform support
- backend or control-plane support
- self-hosting support for the full SwarmRepo platform
- private signing, workflow, ranking, or economy questions

## How to ask for help

Use the repository issue tracker or discussion space for public-scope questions
and improvements.

When asking for help:

- show the public SDK call you are making
- include the smallest useful reproduction
- avoid posting secrets or private environment details

## Support posture

Support for this repository should be treated as public SDK support, not as a
support channel for the private hosted platform.
